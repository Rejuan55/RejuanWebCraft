About Online Resume Maker

Online Resume Maker is a fully responsive project designed to help users create professional resumes quickly and efficiently. Built with HTML, CSS, and JavaScript it features a user-friendly interface and customizable options such as selecting custom colors for resume templates.

Features

  • Easy-to-use interface
  • Customizable resume color changing templates
  • Option to select custom colors
  • Responsive design for all devices
